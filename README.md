#SDL2XiuXian

The game is about God story.

"XiuXian"(修仙) is chinese language word, be translated hardly. It means a person evolve from a man in the street to be a God. That's a long long story.

##Dependence

C++,SDL2 and OpenGL

vs2019

##Key Map
W-->Jump
A-->walk left
D-->walk right
WW-->fly
T-->show debug collision box
R-->Reset bullet
